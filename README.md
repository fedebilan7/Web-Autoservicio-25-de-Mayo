# Autoservicio 25 de Mayo - Landing Page

Este proyecto es una Landing Page moderna y responsiva desarrollada para un comercio minorista real ubicado en Moreno, Buenos Aires. El objetivo es digitalizar la presencia del autoservicio, permitiendo a los clientes consultar ofertas y la ubicación del local de forma sencilla desde cualquier dispositivo.

## 🚀 Tecnologías Utilizadas

* **HTML5**: Estructura semántica del sitio.
* **CSS3**: Diseño personalizado, grillas (CSS Grid) y adaptabilidad móvil (Media Queries).
* **JavaScript**: Lógica para la interactividad del menú desplegable (hamburguesa) en dispositivos móviles.

## ✨ Características

* **Diseño Responsivo**: Optimizado para celulares, tablets y computadoras.
* **Sección de Ofertas**: Grilla dinámica para destacar productos con descuentos especiales.
* **Integración con Google Maps**: Mapa interactivo para facilitar la llegada de los clientes.
* **Acceso Directo a WhatsApp**: Botón de contacto rápido para pedidos o consultas.

## 🛠️ Instalación y Uso

1. Clona el repositorio:
   `git clone https://github.com/fedebilan7/Web-Autoservicio-25-de-Mayo.git`
2. Abrí el archivo `index.html` en tu navegador preferido.
